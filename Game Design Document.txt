遊戲設計規範 (GDD / Spec Document)
1. 核心屬性系統 (Primary Stats)
遊戲基礎屬性採用 1-100 區間（初始值可突破），其對戰鬥數據的影響公式如下：

臂力 (Brawn)：影響物理攻擊力與外門防禦。

計算：pAtk += brawn * 2；fixDef += brawn * 1.5。

根骨 (Physique)：影響生命值上限與外門防禦。

計算：maxHp += physique * 25；fixDef += physique * 2。

內息 (Qi Capacity)：影響行動速度、護體減傷與環境道具上限。

計算：atbSpd += qiCap * 1.5；pctDef += (qiCap * 2) / 10。

真元 (Qi Potential)：影響內功威力。

計算：qAtk += qiPot * 2 + qiCap * 1.5。

身法 (Agility)：影響閃避、行動速度與物理攻擊。

計算：dodge += agi * 2；atbSpd += agi * 2。

靈巧 (Dexterity)：影響命中、連擊上限與追擊機率。

計算：comboMax += dex * 2；追擊率 = dex / 200。

洞察 (Perception)：影響命中、爆擊率與閃避。

計算：hit += per * 2；critChance += (per * 2) / 10。

悟性 (Comprehension)：影響內功修練成功率。

計算：art_taiji 修練成功率受悟性加成。

福緣 (Luck)：影響爆擊率與爆擊倍率。

計算：critMult = 1.5 + (luck * 2 + brawn * 1.5) / 100。

2. 戰鬥機制邏輯 (Combat Mechanics)
2.1 ATB 與行動條
累積邏輯：每 50ms 增加 atbSpd / 20 的等待值。

行動門檻：等待值達 100 時進入行動回合，並獲得等同於 comboMax 的氣力值。

2.2 傷害計算公式
原始傷害：(基礎攻擊 + 招式威力) * 隨機波動(0.9~1.1) * 反應倍率。

防禦減免：(原始傷害 - 固定減傷 fixDef) * (1 - 比例減傷 pctDef / 100)。

註：比例減傷 pctDef 最高上限為 80%。

2.3 連擊與破綻 (Combo Chain)
連段消耗：施展招式需扣除對應的 comboCost。

破綻判定：每次連段需進行隨機骰值 (1-100)，若骰值大於「剩餘氣力值」，則發生破綻強制結束回合。

3. 標籤反應系統 (Tag Reaction Guide)招式標籤 (Tags) 是觸發深度連鎖的核心，定義如下：標籤預期效果典型反應 (Reaction)寒疊加寒氣印記疊滿 3 層轉為「冰封」，防禦歸零且無法行動炎疊加炎勁印記與「寒」觸發「陰陽相激」，造成 300 點真傷及 1.5 倍威力鈍具備重擊屬性攻擊「冰封」目標觸發「冰封碎裂」，造成 15% 最大生命值傷害風具備擴散屬性若場上有「火種」觸發「風火燎原」，引發火海追加傷害銳具備穿透屬性若目標有「寒氣」觸發「冰刃刺骨」，追加真傷並提升 1.2 倍威力牽引控制位移屬性吸引環境中的「暗器」攻擊敵方，觸發「萬物歸宗」絲線束縛印記疊滿 5 層觸發「天羅地網」，造成 300 點真傷

4. 內功與經脈規範 (Internal & Meridian)
修練消耗：每次衝穴消耗 100 修為 (Exp)。

成功率計算：基礎率(40) + 屬性加成 - (已通穴位 * 10)。

例如：純陽功受臂力與根骨加成。

境界加成：內功處於「運轉中」時，每一重境界都會依照 db_internal 定義提供屬性加成。當內功鍛鍊到最高境界時，會獲得一定程度的屬性加成。

5. AA 視覺規範 (VFX & Avatar Spec)
特效行為：所有招式粒子必須對應 db_vfx 的四種行為：projectile (彈道)、rain (雨落)、explode (爆炸)、float_up (漂浮)。

立繪分層：

第一層：aa-layer-body (素體，隨裝甲材質變色)

第二層：aa-layer-weapon (武器，隨裝備更換位置與形狀)
